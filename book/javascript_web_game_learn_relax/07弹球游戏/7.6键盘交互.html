    # 7.6 键盘交互

    键盘交互同样适用jQuery来实现，像之前一样将jQuery的js文件添加到网页中后，利用以下代码检测按键时间。

    $("body").keydown(function(event) {

});

    大家能够看到这和之前鼠标事件检测的代码很像，只是鼠标事件检测中的函数式click，而键盘事件检测中的函数是keydown。另外两个事件对应的对象也不同，鼠标事件对一个的是画布canvas，而键盘事件对应的是网页当中的标签项<body>。将以下代码输入网页代码中。

    $("body").keydown(function(event) {
    console.log(event.keyCode);
});

    在按键实践中，我们每按下一个键，就会产生一个按键事件，而具体的按键信息则存放在keyCode当中，这里我们将这个keyCode在控制台中显示出来（见图7.6），以找到左右箭头对应的值。

    • 图7.6 查看按键的值]

    通过测试，我们得到左键对应的值是37，而右键对应的值是39.有了这两个值之后，我们就可以通过判断按键值来控制挡板移动了，代码如下。

    $("body").keydown(function(event) {
    console.log(event.keyCode);
    if (event.keyCode === 37) {
        panel.x = panel.x - 5;
        if (panel.x < 0) {
            panel.x = 0;
        }
    };
    if (event.keyCode === 39) {
        panel.x = panel.x + 5;
        if (panel.x > (400 - panel.xSize)) {
            panel.x = 400 - panel.xSize;
        }
    }
});

    目前对于这个弹球游戏来说，就剩下挡板和小球的碰撞检测了。我们需要修改一下之前小球的边缘检测方法。这个方法实现的功能是： （1）小球碰到上边、左边以及右边时和之前的状态一样，改变运动方向； （2）小球碰到下面时要判断挡板是不是在对应的位置，如果在则改变运动方向，如果不在则提示游戏结束； （3）游戏结束后将小球恢复到初始位置。

    为了实现上述功能，我们为边缘检测方法添加了两个参数——挡板的起始位置panelStart和挡板的结束位置panelEnd，完成后代码如下。

    ball.checkCanvas = function (panelStart, panelEnd) {
    if (this.x < this.r || this.x > 400 - this.r) {
        this.xSpeed = -this.xSpeed;
    }
    if (this.y < this.r ) {
        this.ySpeed = -this.ySpeed;
    }
    if (this.y > 390 - this.r) {
        if (this.x > panelStart && this.x < panelEnd) {
            this.ySpeed = -this.ySpeed;
        } else {
            alert("Game Over!!");
            this.x = 50;
            this.y = 100;
        }
    }
};
