# 2.6.2 逻辑运算符

数字有加减乘除运算符，这些运算符能够将数字组合起来进行运算。而布尔类型的数据有逻辑运算符，逻辑运算符能够计算真或假这样的数据，最终再得到一个真或假的布尔类型值。

JavaScript或者说绝大多数编程语言中都包含3个主要的逻辑运算符，它们是&&、||和！，分别叫作与、或、非。

与操作用来判断两个布尔类型的值是不是都为真，只有两个值都为真时结果才为真；如果其中一个值为假，结果就是假。这样的关系可以用表2.2表示。

表2.2 与操作

条件1 条件2 结果
真 假 假
假 真 假
假 假 假
真 真 真

上表在代码中表示如下。

> true && true
< true
> true && false
< false
> false && true
< false
> false && false
< false

逻辑运算通常会用在判断语句中，可能大家觉得这种运算很奇怪，不过随着本书的展开，通过具体的项目、例子，大家就会发现逻辑运算的好处。

或操作用来判断两个布尔类型的值是不是都为假，只有两个值都为假，结果才为假；如果其中一个为真，那么结果就是真。这样的关系可以用表2.3表示。

表2.3 或操作

条件1 条件2 结果
真 假 真
假 真 真
假 假 假
真 真 真

而非操作就是取反，使用时只需要一个条件，运算符放在条件之前，条件为真，则结果为假；条件为假，则结果为真。或、非两个逻辑运算符在代码中的使用形式如下。

> false || true
< true
> !true
< false
> !false
< true
