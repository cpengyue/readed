# 4.3.1 输入对话框

首先出场的是输入对话框，函数名是prompt，这个函数后面可以跟一个参数，这个参数是对话框中显示的提示信息。另外，我们在对话框中输入的内容会作为函数发的返回值返回。

将上一节定义的message变量的那一句稍作修改。

<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
        <h1>nille's code life</h1>
        <script>
            var message = prompt("what's your name?");
            console.log(message);
        </script>
    </body>
</html>

保存文件后用浏览器打开，就会看到如图所示的界面。

  • 图4.4 在网页中显示对话框

这个对话框中有一个用来输入信息的文本框，还有两个按钮：“确定”和“取消”。当我们在对话框中输入内容并单击“确定”时，这个内容就会显示在控制台中，如图4.5和图4.6所示。

  • 图4.5 在对话框中输入内容并单击“确定”

  • 图4.6 输入的内容会显示在控制台中

之前说过对话框有两个按钮，当单击“确定”时，能够看到控制台中输出的是我们输入的内容，而当单击“取消”时，控制台中会输入信息“null”，如图4.7所示。

  • 图4.7 单击“取消”时，控制台中会输入信息“null”

这是因为当我们单击“取消”时，函数prompt会返回null（空值）。
