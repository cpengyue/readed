# 4.4.3 替换文本内容

使用innerHTML属性同样能够修改这个一级标题的文本内容，只需要使用赋值语句就可以了。现在我们来完成之前描述的功能：通过弹出的输入对话框来改变页面中显示的文字内容。

修改4.4.1节中的文本内容如下。

<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
        <h1 id="heading">nille's code life</h1>
        <script>
            var heading = document.getElementById('heading');
            console.log(heading.innerHTML);
            var message = prompt("what's your name?");
            heading.innerHTML = "hello " + message;
            console.log(heading.innerHTML);
        </script>
    </body>
</html>

  • 图4.13 输入的信息会显示在页面中

在JavaScript代码中，我们首先创建了一个变量heading来指定id为“heading”的对象，接着就在控制台中通过innerHTML属性将对象的文本内容显示出来，在图4.12中我们能够看到此时的文本内容是“nille's code life”。接着弹出一个输入对话框，询问“what's your name?”我们再对话框中输入的信息会返回给变量message，这个变量的值会结合字符串“hello ”赋值给id为“heading”的一级标题，此时在网页中就能看到这个一级标题的新内容了。代码中，我们还将这个新内容同时显示到了控制台中。

DOM实际上是以面向对象方式描述的文档模型，它定义了表示和修改文档所需的对象、这些对象的行为和属性以及这些对象之间的关系。我们千万不要以为DOM只是一种让JavaScript在浏览器间可移植的方法，实际上DOM的应用已经远远超出了这个范围。
