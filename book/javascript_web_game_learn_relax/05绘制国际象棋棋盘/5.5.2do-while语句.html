# 5.5.2 do-while语句

do-while语句用来实现“直到型”循环，其特点是先执行循环体，然后判断循环条件是否成立，其一般形式为：

do
    语句
while (表达式);
1
2
3

do-while语句流程图如图5.8所示。

功能描述：do-while循环为“直到型”循环，它先执行循环体中的语句，然后再判断表达式是否为真，如果为真则继续循环；如果为假则终止循环。因此，do-while循环至少会执行一次循环体语句。

  • 图5.8 o-while语句流程图

注意

do-while语句在使用时除了要注意循环体至少会执行一次的问题外，在使用时还要注意它是以do开始，以while结束的，while(表达式)后的分号不能丢。
