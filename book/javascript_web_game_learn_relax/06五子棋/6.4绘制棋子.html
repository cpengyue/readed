    # 6.4 绘制棋子

    打开控制台之后，当在棋盘上单击鼠标时，就会在控制台中显示响应的x坐标。我们可以利用这个坐标值来绘制棋子。只需要以坐标位置为原型绘制一个圆就可以了。整体的代码如下。

    <!DOCTYPE html>
<html>
<head>
    <title>canvastest</title>
</head>
<body>
    <canvas id="canvas" width="400" height="400"></canvas>
    <script src="https://cdn.bootcss.com/jquery/3.4.1/jquery.js"></script>
    <script>
        var canv = document.getElementById("canvas");
        var ctx = canv.getContext("2d");
        ctx.beginPath();
        for (var i = 0; i < 19; i++) {
            ctx.moveTo(10 + i*20,10);
            ctx.lineTo(10 + i*20, 370);
            ctx.moveTo(10, 10 + i*20);
            ctx.lineTo(370, 10 + i*20);
        }
        ctx.stroke();
        $('#canvas').click(function(event) {
            console.log(event.offsetX);
            ctx.beginPath();
            ctx.arc(event.offsetX, event.offsetY, 10, 0, Math.PI*2, false);
            ctx.fill();
        });
    </script>
</body>
</html>

    当我们在棋盘上单击鼠标时，就会在相应的位置出现一个黑色的棋子。不过大家可能会发现一个问题：棋子的位置并不是完全压在横竖两条线的交叉点上，总会偏一点，有些会压在线上，还有些甚至在格子中，完全没有压在线上。

    为了让棋子能够正好放在横竖两条线的交叉点上，我们需要判断鼠标单击时的坐标靠近哪个交叉点。判断之后，将棋子放在对应的交叉点就好。利用一个for循环，和每一条线的位置进行比较，如果和这条线的位置相差10像素以内，那么我们就认为和该条线比较靠近。代码如下。

    $('#canvas').click(function(event) {
    console.log(event.offsetX);
    var arcPosX, arcPosY;
    for (var x = 0; x < 19; x++) {
        if (Math.abs(event.offsetX - (10 + x*20)) < 10) {
            arcPosX = 10 + x*20;
        }
        if (Math.abs(event.offsetY - (10 + x*20)) < 10) {
            arcPosY = 10 + x*20;
        }
    }
    ctx.beginPath();
    ctx.arc(arcPosX, arcPosY, 10, 0, Math.PI*2, false);
    ctx.fill();
});

    我们在代码中定义了两个变量arcPosX和arcPosY来保存最终棋子的坐标，它们的值是和当时作比较的线段的位置值一致的。另外，在进行差值计算时，我们用了abs函数来取绝对值。这里要注意鼠标的x坐标要和竖线作比较，而y坐标要和横线作比较。
