# 6.7.2 落子函数

接下来，我们对之前的代码进行一点修改。这里顶一个一个函数placingStone来完成落子功能。这个函数需要3个参数；落子的颜色、落子的水平位置以及落子的垂直位置（这里的位置是指在哪个交叉点上）。而函数执行完之后会返回下一个落子的颜色（如果落子不成功，则落子颜色不变）。

函数的参数直接放在function后面的括号中，这个参数应该只能在函数中使用。函数的内容就是之前完成的落子部分的程序，直接复制过来就可以了，不过这里要注意根据参数改变代码中的变量。完成后代码如下。

var placingStone = function(stoneColor, stoneX, stoneY) {
    if (matrix[stoneX][stoneY] === 0) {
        writeMessage();
        blorwh = !blorwh;
        ctx.beginPath();
        ctx.arc(10 + stoneX * 20, 10 + stoneY * 20, 10, 0, Math.PI * 2, false);
        if (blorwh) {
            ctx.fillStyle = "blue";
            matrix[stoneX][stoneY] = 1;
        } else {
            ctx.fillStyle = "white";
            ctx.stroke();
            matrix[stoneX][stoneY] = 2;
        }
        ctx.fill();
    }
    return stoneColor;
};
